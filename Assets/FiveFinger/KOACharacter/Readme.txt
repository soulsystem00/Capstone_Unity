
==========================================================================================

A 1 set of units used in serviced games.


5 kinds of character (6 kinds of skins by grade, 6 kinds of monster skins by each grade)

- walking animation
- Attack animation
- Skill animation

Contains the spine project files.

How to use

Select and play the Scene file at Fivefinger / KOACharacter / CharacterScene


If you are using Unity 5 version, use Unity5Version.Zip file.

Please refer to the ReadeMe file in the Unity5Version.Zip file.

In Unity5Version.Zip, there are resources corresponding to Spine 3.6.


Testing completed in Unity 2017.1.5f1.

Testing completed in Unity 2018.1.9f2.

Testing completed in Unity 2019.2.11f1.

Testing completed in Unity 2019.3.15f1.

Testing completed at spine runtime spine-unity-3.8-2020-06-24.

Spine Download: https://github.com/EsotericSoftware/spine-runtimes/tree/3.8/spine-unity

Inquiry mail: fivefingerqa@gmail.com

License: One license per project is required.
