# 캡스톤디자인 유니티
캡스톤디자인으로 진행한 게임에 대한 유니티 프로젝트입니다.  

## 개요  
횡스크롤 방식의 타워 디펜스 게임  
1대1로 진행이 되며  
유닛을 소환하여 상대방의 타워를 부수면 승리하게 됩니다.
